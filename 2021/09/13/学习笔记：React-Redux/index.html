Redux 单向数据流

基本概念

  • store state
  • action
  • reducer

概述

  • dispatch(action)
  • reducer -> newState
  • subscribe 触发通知

基本使用

  • 创建一个 reducer (state, action) => state
  • 遵循不可变值的原则，state 变化时要返回一个全新的对象
  • 使用 createStore() ，传入 reducer 作为参数创建 store
  • 订阅更新
  • 修改时通过 state.dispatch() dispatch 一个 action
  • reducer 接收到 action 并判断执行何种操作
